My Projects

Reading Tracker (December 2024 - January 2025)

This project is a personal favourite of mine. I love reading, and I track the books I read in a word document. I thought it would be a great idea to test my full-stack capabilities by creating an app that could do it for me. The front-end is JavaScript, with DOM manipulation to alter the HTML for a seamless, single page experience. PHP is used for the backend, complete with user authentication via sessions, and database access to store and retrieve the readings via MySQL. Friends are included as well, and a chat feature is currently in development. Groups will be added too.

PDI Stock API (November 2024 - December 2024)

This project is something that captivated me while I was learning about RESTful APIs. This project uses PHP to call a Python script. The Python script uses the yfinance library to retrieve historical data on the PDI stock. The data is sent back to the PHP code and stored in a database. When called, the API will update the database if necessary and then send the requested data. I learned a lot about how to properly setup an API and handle potential HTTP errors that come with requests.
